# CCS Rate DB — Cross Currency Swap Rate Tracker

> 6개 KRW 통화쌍의 Cross Currency Swap Rate를 날짜별로 누적·조회하는 브라우저 기반 도구

## 개요

매일 수신하는 CCS Rate 데이터를 복사+붙여넣기로 적재하고,
추이 차트·비교 테이블·엑셀 다운로드 기능을 통해 운용보고서 작성에 활용할 수 있는 단일 HTML 앱입니다.
별도 서버나 설치 없이 브라우저만으로 동작하며, 데이터는 브라우저 localStorage에 저장됩니다.

## 대상 데이터

| 통화쌍 | 비고 |
|--------|------|
| USD/KRW | 주요 관리 대상, 운용보고서 활용 |
| AUD/KRW | 누적 저장 |
| EUR/KRW | 누적 저장 |
| JPY/KRW | 누적 저장 |
| GBP/KRW | 누적 저장 |
| CAD/KRW | 누적 저장 |

만기: 1M / 3M / 6M / 1Y / 2Y / 3Y / 4Y / 5Y / 7Y / 10Y

## 주요 기능

### 데이터 입력
- 두 가지 형식 자동 인식
  - **형식 A**: 테너+값이 같은 행 (`1M\t-1.09%\t-1.88%...`)
  - **형식 B**: 테너 단독 행 후 다음 행에 값
- 열 헤더(USD/KRW 등) 포함 여부 무관, HKD 등 7번째 열 이후 자동 무시
- 파싱 미리보기(USD/KRW 기준) 후 저장
- 날짜별 삭제 기능 (잘못 입력 시)
- 기존 엑셀 파일(.xlsx) 업로드로 과거 데이터 일괄 가져오기

### 추이 차트
- 통화쌍·기간(YTD/1M/3M/6M/전체) 선택
- 만기별 토글 (기본값: 1Y·2Y·3Y·4Y·5Y)
- 최신 USD/KRW 1Y·5Y 레벨 및 전일 대비 변동 표시

### 비교 (운용보고서 활용)
- 기준일 vs 비교일 선택 → USD/KRW 1Y·2Y·3Y·4Y·5Y 변동 테이블
- 전체 만기 커브 바차트 (두 시점 비교)
- **엑셀 다운로드** (비교 결과를 .xlsx로 저장)

### 전체 데이터
- 기간 범위 필터 (시작일~종료일 드롭다운)
- 통화쌍별 전체 데이터 조회
- **엑셀 다운로드** / **복사** (엑셀 붙여넣기용 탭 구분 텍스트)

## 데이터 구조

입력 형식 (탭 구분, 6열×10행, % 포함 가능):
```
        USD/KRW  AUD/KRW  EUR/KRW  JPY/KRW  GBP/KRW  CAD/KRW
1M      -1.09%   -1.88%   0.36%    1.77%    -1.19%   0.35%
...
10Y     -0.79%   -1.55%   0.45%    1.19%    -1.13%   -0.10%
```

저장 형식 (localStorage, key: `ccs_rate_db_v1`):
```json
[
  {
    "date": "2026-06-05",
    "data": {
      "USD/KRW": { "1M": -0.0109, "3M": -0.009, ..., "10Y": -0.0079 },
      "AUD/KRW": { "1M": -0.0188, ... },
      ...
    }
  }
]
```

값은 소수 형태로 저장 (-1.09% → -0.0109)

## 사용 방법

### 매일 데이터 입력
1. 메일에서 데이터 복사 (헤더 포함 가능)
2. **입력** 탭 → 날짜 확인 → 붙여넣기
3. **미리보기** → **저장**

### 운용보고서 활용
1. **비교** 탭 → 기준일(보고서 작성일) / 비교일(전월말) 선택
2. 1Y·2Y·3Y·4Y·5Y 변동 테이블 확인
3. **엑셀 다운로드** 또는 기존 엑셀에 수동 입력

### 과거 데이터 가져오기
1. **입력** 탭 하단 → 엑셀 파일 업로드
2. Sheet2 형식 자동 인식 (일자·1M~10Y 컬럼 구조)

## 기술 스택

- Vanilla HTML / CSS / JavaScript
- Chart.js 4.4.0 (추이·비교 차트)
- SheetJS / xlsx 0.18.5 (엑셀 다운로드·업로드)
- 브라우저 localStorage (서버 불필요)
- GitHub Pages 정적 호스팅

## 주의사항

- 데이터는 **해당 브라우저에만** 저장됩니다 (다른 브라우저·기기 간 공유 불가)
- 브라우저 캐시 전체 삭제 시 데이터가 삭제될 수 있습니다
- 전체 데이터 탭의 **엑셀 다운로드**로 정기 백업을 권장합니다
